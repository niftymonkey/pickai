Constraints control which models make the final selection after scoring. They don't affect scores — they filter during the selection phase to enforce diversity or other rules.

## How Constraints Work

When `recommend()` selects the top N models, it uses a **two-pass algorithm**:

1. **First pass:** Walk the scored list in order. Add a model only if it passes all constraints.
2. **Second pass:** If fewer than `limit` models were selected, fill remaining slots from the scored list, ignoring constraints.

This guarantees you always get `limit` results (if enough candidates exist) while preferring constraint-satisfying models.

## Built-in Constraints

### perProvider(max)

Limits how many models from the same provider appear in results. Default: 1 per provider.

```ts
import { perProvider } from "pickai";

// At most 2 models per provider
const results = pick.recommend(Purpose.Quality, {
  constraints: [perProvider(2)],
  limit: 10,
});
```

### perFamily(max)

Limits how many models from the same family appear in results. Models without a `family` field always pass — they aren't counted against any group.

```ts
import { perFamily } from "pickai";

// At most 1 model per family (no two Claude models, no two GPT models, etc.)
const results = pick.recommend(Purpose.Balanced, {
  constraints: [perFamily(1)],
  limit: 5,
});
```

## Combining Constraints

Pass multiple constraints as an array. A model must pass all of them during the first selection pass.

```ts
const results = pick.recommend(Purpose.Quality, {
  constraints: [perProvider(2), perFamily(1)],
  limit: 10,
});
```

## Writing Custom Constraints

A `Constraint` is a function that receives the models already selected and the candidate being considered:

```ts
type Constraint = (selected: Model[], candidate: Model) => boolean;
```

Return `true` to allow the candidate, `false` to skip it (in the first pass).

```ts
// Custom: no more than 1 model costing over $10/M input tokens
const budgetCap: Constraint = (selected, candidate) => {
  if ((candidate.cost?.input ?? 0) <= 10) return true;
  const expensiveCount = selected.filter((m) => (m.cost?.input ?? 0) > 10).length;
  return expensiveCount < 1;
};
```
