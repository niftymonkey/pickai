A **source** is a function with the signature `() => Promise<Model[]>`. `createPicker` calls it once at creation time, then caches the result for all subsequent `find()` and `recommend()` calls.

## fromModelsDev()

The built-in source fetches the full catalog from [models.dev](https://models.dev) — 3,700+ models from 100+ providers.

```ts
import { createPicker, fromModelsDev } from "pickai";

const pick = await createPicker(fromModelsDev());
```

### Pre-fetched Data

If you've already fetched the models.dev data (for caching, offline use, or testing), pass it directly:

```ts
import { fromModelsDev, type ModelsDevData } from "pickai";

const cached: ModelsDevData = JSON.parse(fs.readFileSync("models.json", "utf-8"));
const pick = await createPicker(fromModelsDev(cached));
```

### Data Format

`fromModelsDev` handles all the parsing internally:

- Converts `snake_case` API fields to `camelCase` (`tool_call` → `toolCall`)
- Derives `openRouterId` from provider and model ID
- Resolves `sdk` from each provider's `npm` field
- Handles missing optional fields (`cost`, `knowledge`, `family`, etc.)

## Custom Sources

Any function matching `() => Promise<Model[]>` works as a source. This lets you:

- Maintain your own curated model list
- Load from a database or API
- Combine multiple data sources
- Test with fixed model data

The only requirement is that each model conforms to the `Model` interface.

## parseModelsDevData()

If you want to use the models.dev parser without the fetch wrapper, it's exported directly:

```ts
import { parseModelsDevData } from "pickai";

const data = await fetchFromYourCache();
const models = parseModelsDevData(data);
```

This returns `Model[]` synchronously. Useful when you manage fetching and caching yourself.
